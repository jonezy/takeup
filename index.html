Takeup HTML

Takeup Html is a set of files for starting out on html + css projects. It's currently being used as an example of responsive css. This isn't really meant for general consumption.

The most bare set of css has been included with a primary focus being on reset and grids, a little smattering of typography but that's about it.

Credits

Various portions of this framework come from or are inspired by:
  • Reset.css
  • Skeleton CSS
  • 1140px.com

Reset

The very first thing that we do in the Takeup framework is to reset the default browser styles. The primary reason for this is to have a 'clean slate' when applying further styles. This also helps with cross browser compatibility issues.

For more detailed information go check out Eric Myers page for reset.css

Grids
